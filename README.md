# Flee_WebServer
Flee WebServer是一个高性能webserver，使用epoll实现IO多路复用，ET模式，IO模型为Reactor模式，同时使用多线程充分利用多核CPU，并使用线程池避免线程频繁创建销毁的开销，用了状态机解析HTTP请求，支持GET、POST请求，支持长连接，支持注册登陆。正在逐步更新中。
